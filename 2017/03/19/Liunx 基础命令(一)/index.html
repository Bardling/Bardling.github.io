建立链接

软链接：

  • 软链接以路径的形式存在。类似于Windows操作系统中的快捷方式
  • 软链接可以 跨文件系统 ，硬链接不可以
  • 软链接可以对一个不存在的文件名进行链接
  • 软链接可以对目录进行链接

硬链接：

  • 硬链接，以文件副本的形式存在。但不占用实际空间
  • 不允许给目录创建硬链接
  • 硬链接只有在同一个文件系统中才能创建

文件基本属性

文件权限一共10位，第一位属性为

  • 当为 d 则是目录
  • 当为 - 则是文件
  • 若是 l 则表示为链接文档(link file)
  • 若是 b 则表示为装置文件里面的可供储存的接口设备(可随机存取装置)
  • 若是 c 则表示为装置文件里面的串行端口设备，例如键盘、鼠标(一次性读取装置)。

其余九位每三位为一组分别代表所有者、所属组、其他人。

  • r 代表读
  • w 代表写
  • x 代表执行
